# User Community Dashboard API

A simple Node.js + Express API exposing users and communities endpoints.

## How to run the project
1. Install dependencies
   - cd d:\zedd_assign\user-community-dashboard-api
   - npm install
2. Start the server
   - npm start
3. Default server
   - URL: http://localhost:4000/
   - Endpoints:
     - GET /users
     - POST /users
     - GET /communities/:id

## How to run tests
- Run the full test suite
  - npm test
- Optional: run Jest in watch/verbose
  - npx jest --watch
  - npx jest --verbose

## Time taken
- toatal-50 minutes
